# MobileApp_Codex

A Flutter grocery price comparison app concept based on a multi-store savings workflow.

## What is included

- Category-first grocery browsing
- Product search and side-by-side comparison
- Cart total comparison across stores
- Store information and convenience highlights
- Alerts and premium features mockup
- Clean file structure with mock data ready for backend integration

## Project structure

- `lib/main.dart` entry point
- `lib/core` theme setup
- `lib/data` mock stores, categories, and product pricing
- `lib/models` app data models
- `lib/screens` main app screens
- `lib/widgets` shared shell/navigation

## Screens

- Home
- Search & Compare
- Cart Comparison
- Stores
- Profile & Alerts

## Next recommended steps

1. Install Flutter SDK on your machine
2. Run `flutter pub get`
3. Run `flutter run`
4. Replace mock data with a repository layer and real APIs
5. Add state management, authentication, and backend sync

## Product direction

This version is optimized as an MVP prototype for:

- budget-conscious shoppers
- families comparing full-cart totals
- deal and alert driven grocery savings workflows
